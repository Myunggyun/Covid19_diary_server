## Corona-Diary

### 소통을 통해, 공감대를 형성하고, 더 나아가 빠른 회복을 위한 일기

# Commit Rule

1. 커밋전에 롤 첵크 필수 하기
2. 머지 전에 코드리뷰 받기
3. eslint:recommended 따르기
4. 커밋 메세지는 한글로 작성하기
5. 커밋 메세지 => "이모지 [커밋목적] 커밋제목"
6. 이모지 =>  
   :books: [패키지 설치]
   :sparkles: [추가]
   :hammer: [수정]
   :bronken heart: [버그]
   :nail_care: [CSS]
   :guitar: [기타]
